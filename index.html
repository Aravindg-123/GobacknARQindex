What is Go-Back-N ARQ?

Go-Back-N ARQ (Automatic Repeat reQuest) is a sliding window protocol used for reliable data transmission in computer networks. It allows multiple frames to be sent before requiring an acknowledgment, improving network efficiency.

Key Features:

  • Sliding Window: Sender can transmit up to N frames before waiting for acknowledgment
  • Cumulative ACKs: Receiver acknowledges the last correctly received frame in order
  • Error Recovery: On timeout, sender retransmits from the base frame onwards
  • Efficiency: Improves throughput by utilizing pipeline transmission
